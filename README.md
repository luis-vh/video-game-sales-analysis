# Proyecto de Análisis de Ventas de Videojuegos

## Descripción
Este proyecto tiene como objetivo analizar un dataset público de ventas de videojuegos utilizando **Python, SQlite y Excel**, partiendo de un archivo csv.

## Tecnologías utilizadas
- **Python**: para limpieza de datos y conexión con SQLite.
- **SQLite**: para almacenar el dataset y realizar consultas SQL.
- **Pandas**: para manipulación de datos y exportación a Excel.
- **Excel**: para visualisacion más clara y concisa.

## Flujo de trabajo
1. **Documentación**: se creó este README para explicar el proceso y los hallazgos.
2. **Carga del  dataset**: se cargó el archivo csv a una base de datos SQLITE.
3. **Consultas SQL**: se realizaron consultas para responder preguntas clave.
4. **Exportación de resultados**: los resultados se eportarpn a Excel.
